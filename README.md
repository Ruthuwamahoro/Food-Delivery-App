# Food Delivery App

This App is designed to help users get easy support for ordering the food online and get it on time.It has a all the backend features with its connected frontend functionality.

# Tech stack

** Java SpringBoot
** MongoDb
** Cloudinary

# API Endpoints

- /api/users
- /api/roles
- /api/foods
- /api/carts
- /api/carts/items
- /api/orders
